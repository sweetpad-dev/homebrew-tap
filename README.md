# homebrew-tap

Homebrew tap for the [sweetpad](https://github.com/sweetpad-dev/sweetpad) CLI — *xcodebuild for humans*.

## Install

```sh
brew install sweetpad-dev/tap/sweetpad
```

`Formula/sweetpad.rb` is updated automatically by CI on each `cli-v*` release of the main repo; the binary is a Developer ID-signed, notarized universal macOS build.
